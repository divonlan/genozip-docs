.. option:: -T, --files-from filename.  An alternative to providing input file names on the command line. filename it a textual file containing a newline-separated list of files. If filename is - (a hyphen) data is taken from stdin rather than a file.

                     |

.. option:: --log filename.  Send non-file output to a log file instead of the terminal.

                     |

.. option:: --echo  Output the full command line upon successful or failed completion of execution.

                     |

.. option:: -h, --help  Show a link to this page.

                     |

.. option:: --help=attributions  Show attributions.

                     |

.. option:: -L, --license, --licence  Show the license terms and conditions for this product as accepted. Combine with --force to see the most up-do-date version of the license. If you wish to change your license to the most recent one - re-register with genozip --register. 

                     |

.. option:: -V, --version  Display Genozip's version number.

                     |
